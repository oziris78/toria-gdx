## Importing ToriaGDX
Add these to your build.gradle file

```GROOVY
allprojects {
    repositories {
        maven { url 'https://jitpack.io' }
    }
}

dependencies {
    implementation 'com.github.oziris78:toria-gdx:v0.1.0'
}
```
