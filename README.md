## Importing TelekGDX
Add these to your build.gradle file

```GROOVY
allprojects {
    repositories {
        maven { url 'https://jitpack.io' }
    }
}

dependencies {
    implementation 'com.github.oziris78:telek-gdx:v0.0.10'
}
```
